WHAT WE CAN'T GET (AND WHY)
================================================================================

Individual probability predictions for any player other than the logged-in user:
  • Need: probability-specific session JWT (separate from main SportsPredict JWT)
  • Where it lives: sessionStorage of the probability cup SPA
  • Block: Chrome extension security policy prevents reading sessionStorage
  • Fallback: Playwright can't log in without credentials
  • The main SportsPredict JWT (we have it) does NOT work for /probability/* endpoints

To unlock this, would need one of:
  A) User provides their Google account credentials for Playwright to log in
  B) User manually copies their session token from browser dev tools console:
     Run in console: JSON.stringify(Object.entries(sessionStorage))
     → This would reveal the probability session token
