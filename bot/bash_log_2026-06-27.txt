--------------------------------------------------------------------------------
INPUT:
  cd ".../bot" && python3 -c "
  settled = [m for m in markets if m['status'] == 'settled']
  vals = Counter(m['current_value'] for m in settled)
  for v, count in sorted(vals.items()): print(f'{v}: {count} markets')
  ambiguous = [m for m in settled if m['current_value'] not in (0, 100)]
  print(f'Markets settled with non-0/100 current_value: {len(ambiguous)}')
  "

OUTPUT:
  current_value distribution on settled markets:
    0: 367 markets
    100: 282 markets

  Markets settled with non-0/100 current_value: 0

  CONFIRMED: current_value is ALWAYS 0 (NO) or 100 (YES) after settlement.
  It does NOT encode crowd probability — only binary outcome.
  Crowd's final probability estimate is not stored in this API field.
