## Known gaps found while building this folder

- **Venue for this semifinal is NOT locally determinable.** Checked `bot/data/matches.json`, `bot/data/all_match_details.json` (81 matches, none match this fixture — venue field not populated for any entry in that file), `bot/data/match_questions/ENG_vs_ARG.json` (no venue field), `data/international_results/results.csv` (has no 2026-07-15 row at all — only through the group stage for 2026), `data/processed/espn_match_panel.csv` / `data/processed/espn_raw_events/` (both stop at the QF, event_id 760513; no event 760514/760515 fixture file exists for this match — 760514 is France-Spain's SF, the sibling SF), and the Smarkets snapshot (corners/cards markets only, no venue metadata). Unlike the France-Spain folder, which had a live ESPN SF-fixture pull (`espn_FRA_ESP_sf_fixture_760514.json`) confirming AT&T Stadium, no equivalent pre-match ESPN fixture pull exists yet for England vs Argentina anywhere on disk. **This needs a live fetch — flagged as the top open gap for the next step**, since it blocks `06_wc2026_venue_altitude_reference.csv` from being narrowed to a single relevant row and blocks any altitude/travel-fatigue reasoning.
- **Transfermarkt player-level roster (`national_team_players.csv` / `national_team_player_valuations.csv`) returns zero rows for both England and Argentina**, whether joined on `current_national_team_id` (3299/3437) or on `country_of_citizenship` — verified directly (0 rows both ways for both teams). This is the identical gap already documented for France and Spain in the sibling MANIFEST — England and Argentina are also simply absent from this particular Transfermarkt extract, despite being two of the biggest, most valuable squads. Not included in this folder; flag if individual player market-value data is needed for Messi/Kane/Bellingham/Álvarez-type questions — will need a different source.
- **`rest_days_features.csv` is upstream-stale** (per `MASTER_DATASET_DICTIONARY.md`, "stops 2026-06-09, before the tournament even started") — included here only for parity with the France/Spain folder's file list; not reliable for this match's actual rest-day computation without a fresh recompute from `07_all_matches_either_team.csv`'s real fixture dates.
- No ESPN pre-match rosters/last-5-games form compilation, no full Smarkets quote set (match-winner/goals/BTTS/player props), and no JTC crowd consensus are in this folder yet — all of these require a live fetch/research pass, not a duplication of existing local data, and are the next step (mirrors the France/Spain folder's own "Known gaps" note before its pricing session).
- `05_team_venue_distances.csv` and `06_wc2026_venue_altitude_reference.csv` are both **generic team↔any-venue / all-venue reference tables**, not specific to this match's actual venue (same shape as the France/Spain versions) — so the venue gap above does not block producing these two files, only their eventual narrowing/use in modeling.
